## Commands:

- Start a new project with Vite:

```bash
npm init vue@latest
```

- Go to your new project folder:
```bash
cd <your-project-name>
```

- Install dependencies and run it locally at [http://localhost:5173/](http://localhost:5173/):

```bash
npm install
npm run dev
```
